Currency Converter API


Which API?
I chose a currency converter for my API. You can enter a value and calculate how much it translates to in an other currency. My code works for about 40 currencies (I didn't want to go overboard).
Website
website: https://www.currencyconverterapi.com
Options?
As for options, each calculation takes two user-entered currencies and enters them as parameters in a hyperlink for an API call. That API call returns a JSON file/text with the given conversion rate between just those two currencies.

The code then multiplies the conversion rate by the number that the user enters, giving the final answer.
Applications?
This API is very useful for travelers who need to know how much money they will need in a given country. It gives them a quick conversion between their native currency and another country's.

This API could also be useful for economists trying to understand a nation's economic health. The price of one currency versus another can indicate the strength of a nation's economy, as well as potential trade deals.
